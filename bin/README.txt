MPV Binary Folder
=================

Place mpv.exe here for Electron video playback.

Download from:
- https://sourceforge.net/projects/mpv-player-windows/files/
- Or: choco install mpv, then copy from installation folder

Required files:
- mpv.exe (main executable)
- Any required DLLs (copy from MPV distribution if needed)

The Electron app will automatically use this binary when running.
